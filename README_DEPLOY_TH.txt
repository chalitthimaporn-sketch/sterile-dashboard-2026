Sterile Factory Dashboard Website

วิธี Deploy ด้วย Vercel
1. เข้า https://vercel.com
2. สมัคร / Login
3. กด Add New > Project
4. อัปโหลด ZIP นี้ หรือแตก ZIP แล้วอัปโหลดโฟลเดอร์
5. กด Deploy
6. จะได้ URL สำหรับเปิดจากมือถือ

วิธี Deploy ด้วย Netlify
1. เข้า https://netlify.com
2. Login
3. ไปที่ Sites
4. ลากโฟลเดอร์หรือ ZIP นี้ไปวาง
5. จะได้ URL เปิดใช้งานทันที

วิธีใช้งาน
1. เปิด URL Dashboard
2. กด Upload Excel
3. เลือกไฟล์ Excel ล่าสุด
4. Dashboard จะคำนวณใหม่ทันที

หมายเหตุ
เวอร์ชันนี้เป็น Realtime แบบ Upload Excel แล้วคำนวณทันทีใน Browser
ถ้าต้องการ Realtime แท้แบบไม่ต้อง Upload ไฟล์ ต้องเชื่อม Google Sheet API หรือ Database เพิ่ม
